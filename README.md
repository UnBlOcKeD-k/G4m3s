# Games Collection 🎮

A collection of games built with Python.

## Chess ♟️

A basic chess game implementation with a command-line interface.

### Features
- Standard chess board setup with all pieces
- Algebraic notation for moves (e.g., `e2 e4`)
- Turn-based gameplay
- Visual board display

### How to Play

1. Run the game:
   ```bash
   python chess.py
   ```

2. Enter moves using algebraic notation:
   - Columns: a-h (left to right)
   - Rows: 1-8 (bottom to top for white, top to bottom for black)
   - Example: `e2 e4` moves a piece from e2 to e4

3. Players alternate turns automatically
